# Plan 04.1-03: Setup Adaptation For Daily, 4H, And Optional 1H

## Objective

Adapt existing setup contracts so Trend Pullback and Volume Breakout can operate
under the three strategy modes without duplicating the same exact rules across
every timeframe.

## Tasks

1. Preserve `daily_only`: Daily handles eligibility and setup detection using
   completed daily bars.
2. Adapt Trend Pullback for MTF: Daily broader healthy trend, 4H pullback and
   recovery as the primary setup, 1H optional entry-readiness confirmation.
3. Adapt Volume Breakout for MTF: Daily structure, 4H primary breakout,
   1H optional confirmation that price is holding and not overextended.
4. Keep Relative Strength Leader as a Phase 5 consumer: Daily broad leadership,
   4H setup-level RS evidence, 1H supporting only.
5. Maintain at most one candidate per symbol.
6. Ensure 1H never creates trades independently and never overrides failed
   Daily, invalid 4H, `RISK_OFF`, stale data, hard rejection, or invalid
   reward/risk.
