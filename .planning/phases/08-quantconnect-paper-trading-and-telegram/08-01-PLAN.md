| Boundary | Description |
|----------|-------------|

| Local config -> deployment contract | Operator configuration controls whether QuantConnect prerequisites are configured, but secrets must stay outside repository files. |
| Operator command text -> external QuantConnect CLI | The repository may render operator-run commands but tests must not execute deployment. |

| T-08-02 | Tampering | Limited Paper config | mitigate | Validate exact Limited Paper caps and require existing Phase 6 allocation, sector, reward/risk, stop, and target checks. |

| T-08-04 | Information Disclosure | QuantConnect credentials | mitigate | Store only env/config key names; tests reject credential-like values and secret-bearing docs examples. |
| T-08-05 | Spoofing | Deployment target | mitigate | Reject any target other than QuantConnect Paper Trading and forbid real brokerage config. |
| T-08-06 | Tampering | LEAN CLI command wrapper | mitigate | Render command metadata only; unit tests prove no subprocess deployment or Live Node start. |
